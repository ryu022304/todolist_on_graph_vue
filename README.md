# todolist_on_graph_vue
TODOリストをグラフ上に動的に配置するWebアプリ

## Description
[以前作成したもの](https://github.com/ryu022304/todolist_on_graph)をVue.jsでリメイクしたもの。  
自身でグラフの軸を設定し、TODOの優先度を可視化することが可能になる。
